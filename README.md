# COVID-19 Data Analysis |

## Description | الوصف
A comprehensive analysis of COVID-19 data using Python and Plotly, including:
- Exploratory Data Analysis (EDA)
- Metrics for confirmed cases, deaths, and recoveries
- Advanced analyses: ratios, active cases, weekly indicators
- Interactive visualizations to show relationships between variables

تحليل شامل لبيانات COVID-19 باستخدام Python وPlotly، يشمل:
- التحليل الاستكشافي للبيانات (EDA)
- مؤشرات الحالات المؤكدة، الوفيات، التعافي
- التحليلات المتقدمة: النسب، الحالات النشطة، المؤشرات الأسبوعية
- رسوم بيانية تفاعلية لتوضيح العلاقات بين المتغيرات

## Tools | الأدوات
- Python (pandas, plotly)
- Jupyter Notebook

## How to Use | كيفية الاستخدام
1. Open `COVID19_Analysis.ipynb` in Jupyter Notebook  
2. Run all cells to see analyses and interactive visualizations
